static public CURSOR_LAYER:MovieClip
The default layer for cursor, default value is null, mean to use org.aswing.ASWingUtils.getRootMovieClip. If you set a specified value to it, it will be used to replace org.aswing.ASWingUtils.getRootMovieClip.
