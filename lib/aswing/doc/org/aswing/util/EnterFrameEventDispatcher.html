addEventListener

static public function addEventListener(funcOrLis, obj:Object):Object

addEventListener(listener:Object)
addEventListener(func:Function, obj:Object)
addEventListener(func:Function)
Adds a event listener. If you add event listener by the first way, add the listener of the parameter and return it. You can remove the listener later. But if you added by the last two ways, the returned value is a new object of listener, so when you want to remove it, you should use the returned value.

Parameters

funcOrLisit can be listener instance have the listener method, and can be a function which want to handler the event
objwhich context to run in by the func.

Return

the listener just be added. If you add event listener by function, the return is a new object of listener, if you add event listener by a listener object, the return is listener object itself.
