removeDisplayableComponent

static public function removeDisplayableComponent(com:Component):Void

Removes a registered displayable component when it became not displayable.
You should not call this method at your application programs generally with AsWing standard component, But if you created your custom component extension, you should note that you must call the destroy method when it was destroied, then it will be automaticlly removed here, but if you did not call destroy or overrided destroy method, you must call this method manually when it become undisplayable.
