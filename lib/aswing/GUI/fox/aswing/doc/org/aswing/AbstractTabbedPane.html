Class org.aswing.AbstractTabbedPane

  • org.aswing.EventDispatcher
    • org.aswing.Component
      • org.aswing.Container
        • org.aswing.AbstractTabbedPane

Description

An abstract class for all Container pane class that have title, icon, tip for every sub pane. For example JAccordion, JTabbedPane.

setSelectedIndex

public function setSelectedIndex(i:Number):Void

Sets the selected index for this tabbedpane. The index must be a valid tab index or -1, which indicates that no tab should be selected (can also be used when there are no tabs in the tabbedpane). If a -1 value is specified when the tabbedpane contains one or more tabs, then the results will be implementation defined.
