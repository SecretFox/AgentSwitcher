Class org.aswing.RepaintManager

  • org.aswing.EventDispatcher
    • org.aswing.RepaintManager

Description

RepaintManager use to manager the component's painting.

If you want to repaint a component, call its repaint method, it will regist itself to RepaintManager, when frame ended, it will call its paintImmediately to paint its. So component's repaint method is fast. But it will not paint immediately, if you want to paint it immediately, you should call paintImmediately method, but it is not fast.
