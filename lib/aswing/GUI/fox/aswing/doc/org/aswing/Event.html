Class org.aswing.Event

  • org.aswing.Event

Description

All Events are constructed with a reference to the object, the "source", that is logically deemed to be the object upon which the Event in question initially occurred upon. The type is the event's type, generally it is same to the event's handler function name.

Different Events has different extra properties. Extra properties are passed to event handler as additional arguments. The extra properties usually commented in the event name comments.

For example a comments of a Event Type Declaration is: 

 	//onMoved(source:Component, oldPos:Point, newPos:Point)
 	public static var ON_MOVED:String = "onMoved";
It is meaning that this Event's source is a Component instance, its type is "onMoved", and it has extra properties oldPos and newPos, both are Point instances.
