Class org.aswing.awml.AwmlEventListenerInfo

  • org.aswing.awml.AwmlEventListenerInfo

Description

Provides access to the event listener info initialized from the AWML file. Consists of listener which is listener ID registered within AwmlManager using AwmlManager#addEventListener method and method which is belonged to listener and will be called to handle event.
