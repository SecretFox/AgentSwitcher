Constructor Detail

AwmlLoader

public function AwmlLoader(parseOnLoad:Boolean)

AwmlLoader()
AwmlLoader(parseOnLoad)

Constructors new AwmlLoader instance. If parseOnLoad argument is true AWML will be parsed after loading automatically.

Also if parseOnLoad is true action listeners will be invoked after AWML will be parsed. Else action listeners will be invoked once AWML will be loaded.

Parameters

parseOnLoad(optional) the flag specified if AWML needs to be parsed on load or not. Default is true
