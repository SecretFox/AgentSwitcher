Class org.aswing.awml.AwmlManager

Field Detail

STRATEGY_APPEND

static public STRATEGY_APPEND:Number
New component with the existed ID will be appended. Existed component with the same ID will stay.

STRATEGY_REPLACE

static public STRATEGY_REPLACE:Number
New component with the existed ID will replace old one. Old one will be destroyed.

STRATEGY_IGNORE

static public STRATEGY_IGNORE:Number
New component with the existed ID will be ignored.

STRATEGY_EXCEPTION

static public STRATEGY_EXCEPTION:Number
In new component will have already existed ID Exception will be thrown.
